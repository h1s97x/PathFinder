### 核心代码
```typescript
// 视口状态
interface ViewportState {
  scale: number      // 缩放比例
  offsetX: number    // X 轴偏移
  offsetY: number    // Y 轴偏移
}

// 坐标转换
screenToWorld(screenX, screenY) => { x, y }
worldToScreen(worldX, worldY) => { x, y }

// CSS 变换
transform: `translate(${offsetX}px, ${offsetY}px) scale(${scale})`
```
