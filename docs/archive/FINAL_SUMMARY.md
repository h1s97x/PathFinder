## 📖 使用指南

### 基础操作

1. **添加节点**: 点击工具栏 "+" 或按 N 键,然后点击画布
2. **添加边**: 点击工具栏链接图标或按 E 键,依次点击两个节点
3. **移动节点**: 在选择模式下拖拽节点
4. **编辑**: 点击编辑按钮,然后点击节点或边
5. **删除**: 选中节点/边后按 Delete 键

### 路径查询

1. 切换到侧边栏 "算法" 标签
2. 选择算法(Dijkstra/BFS/DFS/TSP)
3. 选择起点和终点
4. 点击 "运行" 查看结果

### 文件操作

- **导入**: 点击 "导入" 按钮,选择 JSON 文件
- **导出**: 点击 "导出" 按钮,保存为 JSON
- **保存**: 点击 "保存" 按钮,保存到本地存储
