## 问题排查

### 如果遇到问题:

1. **依赖安装失败**
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```

2. **端口被占用**
   ```bash
   # 修改 vite.config.ts 中的端口
   server: { port: 3001 }
   ```

3. **类型错误**
   ```bash
   # 重新生成类型
   npm run build
   ```

4. **样式不生效**
   ```bash
   # 清除缓存
   rm -rf .vite
   npm run dev
   ```
