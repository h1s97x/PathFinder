# 快速开始

## 🚀 5分钟上手指南

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000

### 3. 开始使用

#### 方式一: 加载示例数据

首次打开会显示欢迎对话框,选择:
- **简单示例**: 5个节点的基础地图
- **完整示例**: 10个节点的完整校园地图

#### 方式二: 手动创建

1. 点击工具栏的 **"+"** 按钮
2. 在画布上点击添加节点
3. 点击工具栏的 **链接** 按钮
4. 依次点击两个节点创建边
5. 输入边的权重(距离)

### 4. 查询路径

1. 切换到侧边栏的 **"算法"** 标签
2. 选择算法(如 Dijkstra)
3. 选择起点和终点
4. 点击 **"运行"** 查看结果

## 🎯 核心功能

### 图编辑

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 选择模式 | Space | 选择和拖拽节点 |
| 添加节点 | N | 点击画布添加 |
| 添加边 | E | 连接两个节点 |
| 编辑 | - | 点击节点/边编辑 |
| 删除 | Delete | 删除选中项 |

### 算法

- **Dijkstra**: 最短路径查询
- **BFS**: 广度优先搜索
- **DFS**: 深度优先搜索
- **TSP**: 旅行商问题

### 文件操作

- **导入**: 加载 JSON 格式的图数据
- **导出**: 保存为 JSON 文件
- **保存**: 保存到浏览器本地存储

## 📝 示例操作流程

### 查询最短路径

1. 加载示例数据(简单示例)
2. 点击侧边栏 "算法" 标签
3. 选择 "Dijkstra" 算法
4. 起点选择 "图书馆"
5. 终点选择 "操场"
6. 点击 "运行"
7. 查看高亮的最短路径

### 创建自定义地图

1. 点击工具栏 "+" 按钮(或按 N 键)
2. 在画布上点击5次,创建5个节点
3. 点击工具栏链接按钮(或按 E 键)
4. 依次点击节点对,创建连接
5. 输入每条边的权重
6. 点击 "保存" 保存到本地

### 编辑节点信息

1. 点击工具栏编辑按钮
2. 点击要编辑的节点
3. 修改节点名称和描述
4. 点击 "保存"

## 🔧 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl/Cmd + S | 保存 |
| Space | 切换到选择模式 |
| N | 切换到添加节点模式 |
| E | 切换到添加边模式 |
| Delete | 删除选中项 |
| Shift + 点击 | 多选节点 |

## 💡 提示

- 节点可以拖拽移动(选择模式下)
- 点击边可以编辑权重
- 使用搜索栏快速查找节点
- 点击 "自动布局" 优化节点位置
- 数据会自动保存到浏览器
