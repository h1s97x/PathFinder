# 快速开始指南

## 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0 或 yarn >= 1.22.0

## 安装步骤

### 1. 克隆项目

```bash
git clone <repository-url>
cd campus-navigation
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000

## 基础使用

### 创建第一个图

1. **添加节点**
   - 点击工具栏的 "+" 图标
   - 在画布上点击任意位置
   - 节点会自动创建并显示编号

2. **连接节点**
   - 点击工具栏的链接图标
   - 依次点击两个节点
   - 输入边的权重(距离)

3. **编辑节点信息**
   - 点击工具栏的编辑图标
   - 点击要编辑的节点
   - 在弹出的对话框中修改信息

### 查询最短路径

1. 在侧边栏切换到"算法"标签
2. 选择 "Dijkstra 最短路径"
3. 在画布上选择起点(第一次点击)
4. 选择终点(第二次点击)
5. 点击"运行"按钮查看结果

### 算法可视化

- **播放**: 自动执行算法并显示动画
- **暂停**: 暂停当前执行
- **下一步**: 单步执行,查看每一步的状态
- **重置**: 清除当前结果,重新开始

### 保存和加载

1. **保存图**
   - 点击工具栏的保存图标
   - 选择保存位置
   - 文件格式为 JSON

2. **加载图**
   - 点击工具栏的文件夹图标
   - 选择之前保存的 JSON 文件
   - 图会自动加载到画布

## 快捷键

- `Ctrl/Cmd + S`: 保存
- `Ctrl/Cmd + O`: 打开
- `Delete`: 删除选中的节点/边
- `Ctrl/Cmd + Z`: 撤销
- `Ctrl/Cmd + Shift + Z`: 重做
- `Space`: 切换到选择模式
- `N`: 切换到添加节点模式
- `E`: 切换到添加边模式

## 常见问题

### Q: 如何删除节点?

A: 点击工具栏的删除图标,然后点击要删除的节点。或者选中节点后按 Delete 键。

### Q: 如何修改边的权重?

A: 点击工具栏的编辑图标,然后点击边的中点,在弹出的对话框中修改权重。

### Q: 算法执行太快看不清?

A: 在侧边栏的"设置"标签中调整动画速度滑块。

### Q: 如何导入旧版本的数据?

A: 旧版本使用 .dat 格式,需要先转换为 JSON 格式。可以使用项目提供的转换工具。
